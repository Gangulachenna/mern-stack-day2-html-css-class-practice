# 🚆 IRCTC Services UI - Day 2 Assignment

## 📖 Project Overview

This project is a Day 2 HTML & CSS practice assignment inspired by the IRCTC services section. The goal was to improve my understanding of Flexbox, CSS Hover Effects, Responsive Design, and User Interactions.

## 🚀 Features

* Responsive Flexbox Layout
* Service Cards with Icons
* Hover Effects and Animations
* Click Alert Functionality
* Mobile-Friendly Design
* Clean and Organized CSS Structure

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* JavaScript Alert (onclick)

## 📋 Services Included

* Flights
* Hotels
* Rail Drishti
* E-Catering
* Charter Train
* Holiday Packages
* Tourist Train
* Hill Railways
* Gallery

## 🎯 Learning Outcomes

Through this project, I learned:

* Creating responsive layouts using Flexbox
* Using CSS hover effects
* Implementing smooth transformations
* Building reusable card components
* Applying media queries for mobile responsiveness
* Handling click events using HTML attributes

## 💡 Special Features

### Hover Effect

When users hover over a service card:

* Background color changes
* Border color changes
* Card moves slightly upward
* Icon scales smoothly

### Mobile Responsive Design

The layout automatically adjusts for smaller screens using media queries.

### Click Effect

When a user clicks a service card, an alert message appears indicating the selected service.
